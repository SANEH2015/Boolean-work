<h1># Boolean-work</h1>
This app have boolean exercises 

## Run Locally
Clone the project
```bash
  https://github.com/SANEH2015/Boolean-work.git
```
Go to the project directory
```bash
  cd boolean
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run dev
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
